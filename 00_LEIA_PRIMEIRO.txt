💡 DICAS DE OURO:

   Use Modo Manual com projetor (mais visual!)
   Gere gráfico para colocar nos slides
   Explique por que OPT é melhor mas impraticável

════════════════════════════════════════════════════════════════════════════════

🔍 ALGORITMOS EXPLICADOS EM 30 SEGUNDOS:

🔵 FIFO (Simples)
   → Mais antigo sai, novo entra
   → Rápido, mas não é muito inteligente

🟣 LRU (Mais Inteligente)
   → A página não usada há mais tempo sai
   → Baseia-se em localidade temporal
   → Muito melhor que FIFO na prática

🩷 OPT (Ótimo, mas Impossível)
   → Tira a página que vai ser usada mais tarde
   → Teórico: requer saber o futuro!
   → Sempre melhor ou igual aos outros
   → Serve como limite superior

🔷 RANDOM (Baseline)
   → Tira uma qualquer aleatoriamente
   → Para comparar se outra é melhor
   → Resultado imprevisível

❓ PERGUNTAS FREQUENTES:

P: "O simulador funciona sem internet?"
R: Sim! Após a primeira vez que abre, funciona offline. (Chart.js é carregado
   via CDN, mas se falhar, o resto ainda funciona)

P: "Qual navegador devo usar?"
R: Qualquer um moderno: Chrome, Firefox, Safari, Edge. Funciona até em mobile!

P: "Posso modificar o código?"
R: Sim! É tudo em um arquivo HTML. Abra em editor de texto e customize.

P: "Como executo em sala de aula?"
R: Abra index.html em um projetor. Use Modo Manual para demonstração ao vivo.

P: "Preciso de instalação?"
R: Não! Apenas abra o arquivo HTML. Nada para instalar.

⚠️ POSSÍVEIS PROBLEMAS:

❌ "Não carrega a página"
   ✓ Atualize (Ctrl+R ou F5)
   ✓ Use navegador moderno
   ✓ Tente outro navegador

❌ "Gráfico não aparece"
   ✓ Execute Modo Automático primeiro
   ✓ Espere Chart.js carregar
   ✓ Reconecte-se

❌ "Botões não funcionam"
   ✓ Verifique entrada (quadros e sequência)
   ✓ Veja mensagens de erro
   ✓ Siga as instruções em vermelho
